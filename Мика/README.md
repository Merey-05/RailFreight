## Как открыть

Лучше всего запустить простой локальный HTTP-сервер и открыть в браузере:

PowerShell:

```powershell
cd "c:\Users\user\Desktop\Мика"
python -m http.server 8000
```

Затем откройте в браузере:

```
http://localhost:8000/
```

Если Python не установлен, используйте расширение Live Server в VS Code или любой другой локальный веб-сервер.
